### 👋 About me

Hi! I'm **Yaroslav**, 19, a student from Moscow studying to become a data scientist.

I spend most of my time building my own projects — you'll find them here. I'm most at home with tabular data: gradient boosting, tree ensembles, LightGBM. I write in Python and work with pandas and a range of ML libraries.

Always open to feedback and new connections — if you've got something to share or suggest, reach out.
